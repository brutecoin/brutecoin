Brutecoin
=========

Install
-------

Use `pipenv`_:

.. _pipenv: https://github.com/pypa/pipenv

    pipenv install --dev

Code style
----------

Check code style:

    make lint

Type check:

    make typecheck

pre-commit
----------

Use `pre-commit`_:

.. _pre-commit: https://pre-commit.com/#install

Install the git hook scripts:

    pre-commit install
